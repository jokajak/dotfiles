# Hammerspoon Shortcuts

## Hyper key shortcuts

### General
* Hyper + R --> Reload Config
* Hyper + space --> Seal

### Window Management
* Hyper + F --> Next fullscreen step
* Hyper + ⏎ --> Toggle Full Screen
* Hyper + I --> Show
* Hyper + → --> Next right step
* Hyper + ← --> Next left step
* Hyper + ↑ --> Next up step
* Hyper + ↓ --> Next down step

### Monitor Management
* Hyper + ⌘ + → --> Next Monitor
* Hyper + ⌘ + ← --> Previous Monitor
